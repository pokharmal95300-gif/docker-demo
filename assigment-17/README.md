# PySpark Sales Data Analysis

## Project Overview
This project demonstrates PySpark DataFrame operations on a sales dataset.

## Features
- Read CSV data into DataFrame
- Sort products by sales in descending order
- Display top 3 highest-selling products
- Filter products with sales greater than 80,000
- Save filtered data as CSV output
- Run application inside Docker container

## Technologies Used
- Python
- PySpark
- Apache Spark
- Docker
- Java

python app.py
